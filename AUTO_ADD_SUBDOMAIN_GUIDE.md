# 🎯 دليل إضافة Subdomains تلقائياً

## ⚡ الحل السريع (موصى به)

### الطريقة 1️⃣: تشغيل السيرفر بصلاحيات Admin

1. **أغلق السيرفر الحالي** (إذا كان يعمل)

2. **شغل السيرفر بصلاحيات Admin:**
   - افتح **Command Prompt** أو **PowerShell** كمسؤول (Run as Administrator)
   - انتقل لمجلد المشروع:
     ```cmd
     cd C:\Users\acer\Desktop\taskon\dosc
     ```
   - شغل السيرفر:
     ```cmd
     npm start
     ```

3. **الآن عند إضافة شركة جديدة:**
   - سيتم إضافة الـ subdomain تلقائياً لملف hosts ✅
   - لن تحتاج أي خطوات إضافية
   - الدومين سيعمل مباشرة!

---

### الطريقة 2️⃣: استخدام Batch File

1. **اضغط كليك يمين** على ملف `start-as-admin.bat`
2. اختر **"Run as administrator"**
3. سيتم تشغيل السيرفر بصلاحيات Admin تلقائياً

---

## 🔧 إضافة Subdomain يدوياً (إذا لم تعمل الطرق السابقة)

### للدومين الحالي (mosbah):

**في PowerShell كمسؤول:**
```powershell
node add-to-hosts.js mosbah
```

**أو أضف يدوياً:**
1. افتح Notepad **كمسؤول**
2. File → Open → `C:\Windows\System32\drivers\etc\hosts`
3. أضف في نهاية الملف:
   ```
   127.0.0.1 mosbah.taskon.local
   ```
4. احفظ الملف
5. ارجع للمتصفح: `http://mosbah.taskon.local:4000`

---

## 📋 كيف يعمل النظام؟

1. **عند إضافة شركة جديدة** من لوحة التحكم:
   - يتم إنشاء subdomain تلقائياً (مثل: `mosbah`)
   - السيرفر يحاول إضافة `127.0.0.1 mosbah.taskon.local` لملف hosts
   - **إذا كان السيرفر يعمل بصلاحيات Admin:** ✅ ينجح
   - **إذا لم يكن بصلاحيات Admin:** ⚠️ يظهر تنبيه في الـ console

2. **رسائل الـ Console:**
   - ✅ `تم إضافة النطاق لملف hosts بنجاح` → نجح
   - ⚠️ `لم يتم إضافة النطاق تلقائياً` → احتاج صلاحيات Admin

---

## ⚙️ الملفات المستخدمة

| الملف | الوظيفة |
|------|---------|
| `add-to-hosts.js` | سكريبت Node.js لإضافة subdomains لملف hosts |
| `start-as-admin.bat` | Batch file لتشغيل السيرفر بصلاحيات Admin |
| `server.js` | السيرفر الرئيسي (معدّل لإضافة النطاقات تلقائياً) |

---

## 🎯 الخلاصة

### ✅ للتشغيل التلقائي الكامل:
**شغل السيرفر دائماً بصلاحيات Admin** باستخدام أحد هذه الطرق:
- PowerShell/CMD كمسؤول → `npm start`
- `start-as-admin.bat` (كليك يمين → Run as administrator)

### ⚡ بعد ذلك:
- أي شركة جديدة تضيفها ستعمل تلقائياً
- لن تحتاج تعديل ملف hosts يدوياً
- النظام سيتولى كل شيء! 🎉

---

## 🆘 استكشاف الأخطاء

**المشكلة:** الدومين لا يعمل بعد إضافة الشركة

**الحل:**
1. تحقق من Console السيرفر - هل ظهر ✅ أو ⚠️؟
2. إذا ظهر ⚠️: أعد تشغيل السيرفر بصلاحيات Admin
3. أو أضف الدومين يدوياً: `node add-to-hosts.js اسم-الدومين`

---

**💡 نصيحة:** احفظ shortcut لـ PowerShell كمسؤول في سطح المكتب لتسهيل التشغيل!
