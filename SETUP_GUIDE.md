# 🏗️ TASKON - دليل الإعداد والاستخدام

## 📦 الخطوة 1: تثبيت المكتبات (مرة واحدة فقط)

```powershell
npm install
```

## 🚀 الخطوة 2: تشغيل البرنامج

### للتطوير والاختبار:
```powershell
npm run electron
```

## 🏢 أول مرة - إعداد الشركة

عند فتح البرنامج لأول مرة:

1. **املأ بيانات الشركة:**
   - اسم الشركة
   - اسم المدير
   - البريد الإلكتروني
   - كلمة المرور

2. **اضغط "إنشاء الشركة"**

3. **سيتم:**
   - ✅ إنشاء شركتك في قاعدة البيانات
   - ✅ إنشاء حساب المدير
   - ✅ تسجيل دخول تلقائي
   - ✅ الانتقال لصفحة المشاريع

## 👥 إضافة موظفين

### من لوحة التحكم:
1. افتح صفحة "المستخدمين"
2. اضغط "إضافة مستخدم"
3. املأ البيانات:
   - الاسم
   - البريد الإلكتروني
   - كلمة المرور
   - المشروع (اختياري - للربط بمشروع محدد)
   - الصلاحيات

### تسجيل دخول الموظف:
1. يفتح البرنامج
2. يضغط "تسجيل الدخول"
3. يدخل البريد وكلمة المرور
4. سيدخل تلقائياً للمشروع المخصص له

## 📦 الخطوة 3: إنشاء ملف التنصيب

### لإنشاء ملف تنصيب Windows (64-bit):
```powershell
npm run build-win
```

### لإنشاء ملف تنصيب (32-bit):
```powershell
npm run build-win32
```

### لإنشاء كلا النسختين:
```powershell
npm run build-all
```

## 📁 مكان ملفات التنصيب

بعد البناء، ستجد الملفات في مجلد `dist/`:

```
dist/
├── TASKON-Setup-1.0.0.exe     # ملف التنصيب الكامل (Installer)
└── TASKON-1.0.0.exe           # نسخة محمولة (بدون تنصيب)
```

## 🎯 نظام العمل

### 📋 نسخة واحدة للجميع:
- ✅ **المدير:** يدخل كـ Admin - يرى ويدير كل شيء
- ✅ **الموظفون:** يدخلون بحساباتهم - يرون فقط ما لديهم صلاحية له

### 🔐 الصلاحيات:
- **Admin:** صلاحيات كاملة (*)
- **Manager:** صلاحيات محددة
- **User:** صلاحيات محدودة (عرض فقط)

### 🏗️ المشاريع:
- كل شركة لها مشاريع خاصة
- كل مشروع له قاعدة بيانات منفصلة (Collections)
- الموظف يمكن ربطه بمشروع واحد أو أكثر

## 🔧 حل المشاكل

### المشكلة: "Cannot find module 'electron'"
```powershell
npm install
```

### المشكلة: لا أستطيع تسجيل الدخول
- تأكد من البريد وكلمة المرور صحيحين
- تأكد من اتصال الإنترنت
- تواصل مع المدير لإعادة تعيين كلمة المرور

### المشكلة: السيرفر لا يعمل
تأكد من:
- ✅ اتصال الإنترنت (للوصول لقاعدة البيانات)
- ✅ MongoDB URI صحيح في `server.js`

## 📊 معلومات تقنية

- **حجم التنصيب:** ~150-200 MB
- **المنصة:** Windows 7/8/10/11 (64-bit و 32-bit)
- **قاعدة البيانات:** MongoDB Atlas (Cloud)
- **التقنية:** Electron + Node.js + Express
- **نوع النظام:** Client-Cloud (ليس Client-Server المحلي)

## 🚀 نشر البرنامج

1. بناء ملف التنصيب:
   ```powershell
   npm run build-win
   ```

2. اختبار الملف:
   - افتح `dist/TASKON-Setup-1.0.0.exe`
   - نصبه على جهاز آخر
   - اختبر جميع المميزات

3. توزيع البرنامج:
   - شارك ملف `TASKON-Setup-1.0.0.exe` مع الموظفين
   - كل موظف ينصبه على جهازه
   - يسجل دخول بحسابه الخاص

## 🎨 التخصيص

```javascript

```

ثم شغل:
```powershell
npm run setup
```
