# 🏗️ TASKON - دليل الإعداد والاستخدام

## 📦 الخطوة 1: تثبيت المكتبات (مرة واحدة فقط)

```powershell
npm install
```

## ⚙️ الخطوة 2: إعداد قاعدة البيانات (مرة واحدة فقط)

```powershell
npm run setup
```

هذا سينشئ:
- ✅ شركة افتراضية باسم "شركتي"
- ✅ مستخدم admin بالبيانات التالية:
  - البريد: admin@taskon.local
  - كلمة المرور: admin123

## 🚀 الخطوة 3: تشغيل البرنامج

### للتطوير والاختبار:
```powershell
npm run electron
```

## 📦 الخطوة 4: إنشاء ملف التنصيب

### لإنشاء ملف تنصيب Windows (64-bit):
```powershell
npm run build-win
```

### لإنشاء ملف تنصيب (32-bit):
```powershell
npm run build-win32
```

### لإنشاء كلا النسختين:
```powershell
npm run build-all
```

## 📁 مكان ملفات التنصيب

بعد البناء، ستجد الملفات في مجلد `dist/`:

```
dist/
├── TASKON-Setup-1.0.0.exe     # ملف التنصيب الكامل (Installer)
└── TASKON-1.0.0.exe           # نسخة محمولة (بدون تنصيب)
```

## 🎯 كيفية الاستخدام

1. **التشغيل الأول:**
   - سيفتح البرنامج تلقائياً
   - سيتم تسجيل الدخول أوتوماتيكياً
   - ستظهر صفحة المشاريع

2. **إنشاء مشروع جديد:**
   - اضغط "إضافة مشروع"
   - أدخل اسم المشروع
   - احفظ

3. **الدخول للمشروع:**
   - اضغط على المشروع من القائمة
   - ستفتح لوحة التحكم الرئيسية

## 🎨 التخصيص

### تغيير اسم الشركة:
عدل في ملف `setup-default-data.js`:
```javascript
companyName: 'اسم شركتك هنا',
```

### تغيير بيانات المستخدم الافتراضي:
عدل في ملف `setup-default-data.js`:
```javascript
username: 'اسم المستخدم',
email: 'email@example.com',
password: 'كلمة المرور',
```

ثم شغل:
```powershell
npm run setup
```

### تغيير الأيقونة:
استبدل ملف `favicon.ico` بأيقونة جديدة (256x256 بكسل)

## 🔧 حل المشاكل

### المشكلة: "Cannot find module 'electron'"
```powershell
npm install
```

### المشكلة: "لم يتم العثور على الشركة"
```powershell
npm run setup
```

### المشكلة: السيرفر لا يعمل
تأكد من:
- ✅ اتصال الإنترنت (للوصول لقاعدة البيانات)
- ✅ MongoDB URI صحيح في `server.js`

### المشكلة: النافذة فارغة
- انتظر 3-5 ثواني حتى يبدأ السيرفر
- أغلق البرنامج وافتحه مرة أخرى

## 📊 معلومات تقنية

- **حجم التنصيب:** ~150-200 MB
- **المنصة:** Windows 7/8/10/11 (64-bit و 32-bit)
- **قاعدة البيانات:** MongoDB Atlas (Cloud)
- **التقنية:** Electron + Node.js + Express

## 🚀 نشر البرنامج

1. بناء ملف التنصيب:
   ```powershell
   npm run build-win
   ```

2. اختبار الملف:
   - افتح `dist/TASKON-Setup-1.0.0.exe`
   - نصبه على جهاز آخر
   - اختبر جميع المميزات

3. توزيع البرنامج:
   - شارك ملف `TASKON-Setup-1.0.0.exe` مع المستخدمين
   - أو استخدم النسخة المحمولة `TASKON-1.0.0.exe`
