# 🤝 نظام التعاملات الخارجية - External Services System

## 📋 نظرة عامة

هذا النظام مصمم لإدارة التعاملات مع الموردين الخارجيين والخدمات المساندة في المشروع، مثل:
- ⛽ موردي الديزل
- 🚚 خدمات النقل
- 🚜 تأجير المعدات
- 👷 العمالة المؤقتة
- 📦 توريد المواد
- 🔧 خدمات أخرى

## ✨ المميزات الرئيسية

### 1️⃣ **إدارة كاملة للتعاملات**
- ➕ إضافة تعامل جديد
- ✏️ تعديل التعاملات الموجودة
- 🗑️ حذف التعاملات
- 📎 رفع مرفقات (فواتير/إيصالات)

### 2️⃣ **إحصائيات تفصيلية**
- 📊 إجمالي عدد التعاملات
- 💰 إجمالي المبالغ المصروفة
- 📅 تعاملات الشهر الحالي
- 🏆 أكثر خدمة طلباً

### 3️⃣ **بحث وتصفية متقدمة**
- 🔍 بحث بالاسم أو الوصف
- 🎯 فلترة حسب نوع الخدمة
- 💳 فلترة حسب طريقة الدفع

### 4️⃣ **تصدير البيانات**
- 📥 تصدير إلى Excel/CSV
- 📄 تقارير شاملة

## 🗂️ البيانات المسجلة لكل تعامل

| الحقل | الوصف | مطلوب |
|------|------|-------|
| **اسم الجهة** | اسم المورد أو مقدم الخدمة | ✅ |
| **نوع الخدمة** | ديزل، نقل، معدات، عمالة، توريد مواد، أخرى | ✅ |
| **تاريخ التعامل** | تاريخ العملية | ✅ |
| **القيمة** | المبلغ بالريال السعودي | ✅ |
| **طريقة الدفع** | نقدي، شيك، تحويل بنكي | ✅ |
| **وصف العملية** | تفاصيل ما تم (مثل: توريد 500 لتر ديزل) | ✅ |
| **اسم المستلم** | الشخص الذي استلم المبلغ | ❌ |
| **ملاحظات** | أي ملاحظات إضافية | ❌ |
| **المرفقات** | صورة الفاتورة أو الإيصال | ❌ |

## 🎨 التصميم

- **تصميم عصري احترافي** مع UI/UX متقدم
- **ألوان مميزة** لكل نوع خدمة
- **Badges ملونة** لطرق الدفع
- **Responsive Design** - يعمل على جميع الأجهزة
- **Animations سلسة** عند التفاعل

## 🔌 API Endpoints

### GET /external-services
جلب جميع التعاملات (مرتبة حسب التاريخ)

### GET /external-services/:id
جلب تعامل واحد بواسطة ID

### POST /external-services
إضافة تعامل جديد (مع دعم رفع الملفات)

### PUT /external-services/:id
تحديث تعامل موجود

### DELETE /external-services/:id
حذف تعامل

### GET /external-services/export
تصدير جميع التعاملات إلى CSV

## 🚀 كيفية الاستخدام

### 1. إضافة تعامل جديد
1. اضغط على زر "➕ إضافة تعامل جديد"
2. املأ البيانات المطلوبة
3. ارفع المرفقات إن وجدت
4. اضغط "💾 حفظ التعامل"

### 2. البحث والتصفية
- استخدم مربع البحث للبحث السريع
- اختر نوع الخدمة من القائمة المنسدلة
- اختر طريقة الدفع للتصفية

### 3. التعديل والحذف
- اضغط ✏️ لتعديل أي تعامل
- اضغط 🗑️ لحذف تعامل (مع تأكيد)

### 4. التصدير
- اضغط "📥 تصدير Excel" للحصول على ملف CSV
