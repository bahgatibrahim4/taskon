# 🖨️ نظام الطباعة وحفظ PDF - سندات الاستلام

## 📋 نظرة عامة

نظام متكامل لطباعة سندات الاستلام وحفظها كملفات PDF مع دعم رفع لوجو الشركة. تم تصميم النظام ليكون عملياً وفعلياً للاستخدام اليومي في المشاريع.

---

## ✨ المميزات الرئيسية

### 🏢 رفع لوجو الشركة
- **رفع سهل**: زر واحد لرفع اللوجو
- **معاينة فورية**: عرض اللوجو بعد الرفع مباشرة
- **حفظ دائم**: يُحفظ اللوجو في `localStorage` للمتصفح
- **استخدام تلقائي**: اللوجو يظهر في جميع السندات المطبوعة

### 🖨️ الطباعة الاحترافية
- **تصميم احترافي**: قالب طباعة مصمم خصيصاً للسندات
- **معاينة قبل الطباعة**: عرض السند كاملاً قبل الطباعة
- **طباعة مباشرة**: زر واحد للطباعة الفورية
- **جودة عالية**: تنسيق A4 بدقة عالية

### 📥 حفظ PDF
- **حفظ فوري**: تحويل السند إلى PDF بضغطة زر
- **اسم ذكي**: الملف يُحفظ باسم رقم السند تلقائياً
- **جودة ممتازة**: صور واضحة ونصوص قابلة للقراءة
- **دعم العربية**: النصوص العربية تُحفظ بشكل صحيح

### ✍️ منطقة التوقيعات
- **3 توقيعات**: المستلم، المسؤول، المدير
- **خطوط واضحة**: خطوط للتوقيع فوق كل اسم
- **تنسيق احترافي**: مساحات مناسبة للتوقيع

## 🚀 دليل الاستخدام

### 📤 رفع لوجو الشركة

#### الخطوات:
1. افتح صفحة سندات الاستلام
2. في أعلى الصفحة، ابحث عن قسم "🏢 لوجو الشركة"
3. اضغط على زر "📤 رفع اللوجو"
4. اختر صورة اللوجو من جهازك (PNG, JPG, SVG)
5. ✅ سيظهر اللوجو فوراً في المعاينة

#### معلومات مهمة:
- ✅ **الأحجام المدعومة**: PNG, JPG, JPEG, SVG
- ✅ **الحجم الموصى به**: 500x200 بكسل
- ✅ **خلفية شفافة**: يُفضل استخدام PNG بخلفية شفافة
- ✅ **الحفظ**: اللوجو يُحفظ تلقائياً ويبقى حتى بعد إغلاق المتصفح

#### حذف اللوجو:
```javascript
// في Console المتصفح:
localStorage.removeItem('companyLogo');
// ثم أعد تحميل الصفحة
```

---

### 🖨️ طباعة السند

#### الطريقة 1: من جدول السندات
1. ابحث عن السند في الجدول
2. اضغط على زر **🖨️** في عمود الإجراءات
3. ستظهر نافذة معاينة الطباعة
4. اضغط على **🖨️ طباعة**
5. اختر الطابعة من نافذة الطباعة

#### الطريقة 2: معاينة متقدمة
1. اضغط على زر 🖨️
2. في نافذة المعاينة:
   - **🖨️ طباعة**: للطباعة المباشرة
   - **📥 حفظ PDF**: لحفظ كملف PDF
   - **✕ إغلاق**: للعودة للصفحة الرئيسية

#### اختصارات لوحة المفاتيح:
- **Ctrl + P**: طباعة (عند فتح المعاينة)
- **ESC**: إغلاق نافذة المعاينة

---

### 📥 حفظ كـ PDF

#### الخطوات:
1. اضغط على زر **🖨️** للسند المطلوب
2. في نافذة المعاينة، اضغط على **📥 حفظ PDF**
3. سيتم تحميل الملف تلقائياً باسم: `سند_استلام_R-001.pdf`

#### مميزات ملف PDF:
- ✅ **دقة عالية**: Scale 2x لوضوح ممتاز
- ✅ **حجم A4**: تنسيق قياسي للطباعة
- ✅ **الصور**: اللوجو والعناصر تُحفظ بجودة عالية
- ✅ **العربية**: دعم كامل للنصوص العربية
- ✅ **التوقيعات**: مساحات جاهزة للتوقيع اليدوي بعد الطباعة

#### استخدامات ملف PDF:
- 📧 إرساله بالبريد الإلكتروني
- 💾 أرشفته في نظام الملفات
- 📤 رفعه على السحابة (Google Drive, OneDrive)
- 🖨️ طباعته لاحقاً
- ✍️ توقيعه رقمياً

- **@media print**: تنسيق خاص للطباعة
- **CSS Grid**: لتنظيم تفاصيل السند
- **Flexbox**: لتوزيع العناصر

## 📊 إعدادات PDF المتقدمة

### الإعدادات الحالية:
```javascript
const opt = {
  margin: 10,                    // هوامش 10mm
  filename: 'سند_استلام_R-001.pdf',
  image: { 
    type: 'jpeg', 
    quality: 0.98               // جودة 98%
  },
  html2canvas: { 
    scale: 2,                    // دقة 2x
    useCORS: true,              // للوجو الخارجي
    letterRendering: true        // للنصوص الواضحة
  },
  jsPDF: { 
    unit: 'mm',                  // وحدة القياس
    format: 'a4',                // حجم الورق
    orientation: 'portrait'      // عمودي
  }
};
```

### تخصيص الإعدادات:
يمكنك تعديل الإعدادات في ملف `receipts.html`:

#### تغيير الهوامش:
```javascript
margin: 15,  // هوامش أكبر
```

#### تغيير حجم الورق:
```javascript
format: 'a5',  // ورق A5 أصغر
```

#### الاتجاه الأفقي:
```javascript
orientation: 'landscape'  // عرضي بدلاً من عمودي
```


## 🛡️ الأمان والخصوصية

### حفظ اللوجو:
- ✅ **localStorage فقط**: اللوجو يُحفظ في متصفحك فقط
- ✅ **لا يُرسل للسيرفر**: خصوصية كاملة
- ✅ **Base64**: اللوجو يُحول لنص مشفر
- ⚠️ **الحجم**: احذر من استخدام صور كبيرة جداً (>1MB)

### ملفات PDF:
- ✅ **محلية**: تُحفظ على جهازك مباشرة
- ✅ **لا تُرسل للسحابة**: إلا إذا رفعتها يدوياً
- ✅ **قابلة للطباعة**: بدون قيود DRM

## 🔍 استكشاف الأخطاء

### ❌ المشكلة: اللوجو لا يظهر بعد الرفع
**الحل:**
```
1. تأكد أن الملف صورة (PNG/JPG)
2. تحقق من حجم الصورة (<5MB)
3. جرب مسح localStorage:
   localStorage.removeItem('companyLogo');
4. أعد تحميل الصفحة وحاول مرة أخرى
```

### ❌ المشكلة: PDF لا يتم حفظه
**الحل:**
```
1. تأكد من الاتصال بالإنترنت (لتحميل مكتبة html2pdf)
2. تحقق من Console للأخطاء (F12)
3. جرب متصفح آخر (Chrome موصى به)
4. تأكد من تفعيل JavaScript
```

### ❌ المشكلة: الطباعة تظهر بشكل غريب
**الحل:**
```
1. استخدم معاينة الطباعة أولاً
2. تأكد من اختيار حجم الورق A4
3. اضبط الهوامش على "Default"
4. أزل "Headers & Footers" من إعدادات الطباعة
```

### ❌ المشكلة: النصوص العربية تظهر مقلوبة في PDF
**الحل:**
```
1. تأكد من استخدام خط Cairo (مدمج تلقائياً)
2. تحقق من أن direction: rtl مفعّل
3. استخدم Chrome أو Firefox (دعم أفضل للعربية)
```
